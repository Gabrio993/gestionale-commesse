# Gestionale Ore

Questo progetto è un piccolo gestionale web sviluppato in PHP su base CodeIgniter.

L'applicazione nasce per gestire in modo semplice:

- utenti e ruoli
- clienti
- commesse
- registrazione delle ore lavorate
- report riepilogativi
- esportazione dei dati

## Obiettivo del progetto

Questo repository è anche un esperimento pratico: verificare fino a che punto sia possibile costruire un applicativo funzionante utilizzando quasi esclusivamente strumenti di intelligenza artificiale, con un supporto minimo da parte del programmatore.

L'idea non era soltanto "far generare del codice", ma capire se un flusso di lavoro guidato dall'AI potesse aiutare a:

- impostare la struttura del progetto
- produrre rapidamente nuove funzionalità
- iterare su interfaccia e logica applicativa
- accelerare attività ripetitive o di supporto

Il contributo umano è rimasto comunque importante per validare le scelte, correggere gli errori, rifinire il comportamento dell'applicazione e mantenere una direzione coerente nello sviluppo.

## Stack tecnico

- PHP 8+
- CodeIgniter
- PhpSpreadsheet

## Stato del progetto

Il progetto va considerato come un prototipo evoluto / esperimento reale, più che come un prodotto definitivo. Proprio per questo motivo il codice rappresenta anche un test sul rapporto tra automazione, supervisione umana e sviluppo software assistito dall'AI.
